## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- PHP
- MySQL

## 📦 Features
- User Signup/Login
- Create Post
- View All Posts (Feed)
- Logout

## 🧑‍💻 How to Run Locally

### Backend
- Create MySQL database `linkedin_clone`
- Import tables from `schema.sql`
- Run PHP server: `php -S localhost:8000`

### Frontend
- Open `frontend/index.html` in browser
